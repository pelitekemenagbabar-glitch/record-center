Panduan Penggunaan Aplikasi

Gunakan menu Pencarian Arsip di sidebar untuk menemukan dokumen. Anda dapat mencari berdasarkan nomor arsip, judul, atau tanggal. Filter lanjutan juga tersedia untuk pencarian yang lebih spesifik.

Klik menu Input Arsip Baru. Isi semua kolom yang diperlukan seperti nomor arsip, judul, tanggal, dan lokasi fisik penyimpanan. Unggah file digital jika ada, lalu klik simpan.

Masuk ke menu Laporan untuk menghasilkan rekapitulasi data arsip. Anda dapat memilih periode laporan dan jenis data yang ingin disertakan, kemudian ekspor laporan dalam format PDF atau Excel.
